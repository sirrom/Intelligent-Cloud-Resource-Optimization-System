## 1. Project Overview

This capstone project asks you to build an **end-to-end ML system** that learns
from real cloud-infrastructure telemetry to:

| Goal | Technique |
|------|-----------|
| Predict execution time | Regression |
| Classify workload type | Classification |
| Discover workload groups | Clustering |
| Analyse model generalisation | Bias–Variance / Learning Theory |
| Optimise resource scheduling | Reinforcement Learning |
| Report results holistically | Final Dashboard |

By the end of all six stages you will have trained **15+ models**, produced
**30+ visualisations**, and compiled a **master JSON report** — all inside a
single reproducible notebook.
